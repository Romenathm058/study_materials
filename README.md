# 📚 Short C!rcu!t

A simple and organized platform where students can access **study materials** and **YouTube playlists** for MAKAUT semester exams in one place.

## 🚀 Features

* 📖 Subject-wise study materials (notes, PDFs, important questions)
* 🎥 Curated YouTube playlists for better understanding
* 🗂️ Clean and easy-to-navigate structure
* ⏱️ Focused on exam-oriented preparation
* 📌 Regular updates with new content

## 🎯 Purpose

This project is built to help MAKAUT students prepare efficiently by providing **all essential resources in a single location**, saving time and effort during exams.
